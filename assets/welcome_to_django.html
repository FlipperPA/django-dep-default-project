Poll Application Tutorial

In the official Django tutorial, you will learn the basics about Django by building an application to create polls and allow users to take them. This tutorial picks up right where you are: with Django installed and ready to go.

You will learn the basics of the Django framework project structure, defining models, writing views, and creating templates.

The project you create in this tutorial will consist of two parts:

  • A public site that lets people view polls and vote in them.
  • An admin site that lets you add, change, and delete polls.

Official Documentation

Django's documentation has often been mentioned as one of its best features. It is organized into four basic areas:

  • Tutorials take you by the hand through a series of steps to create a Web application. Start here if you’re new to Django or Web application development. Also look at the “First steps” below.
  • Topic guides discuss key topics and concepts at a fairly high level and provide useful background information and explanation.
  • Reference guides contain technical reference for APIs and other aspects of Django’s machinery. They describe how it works and how to use it but assume that you have a basic understanding of key concepts.
  • How-to guides are recipes. They guide you through the steps involved in addressing key problems and use-cases. They are more advanced than tutorials and assume some knowledge of how Django works.
